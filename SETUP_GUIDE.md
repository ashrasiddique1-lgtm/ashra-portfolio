## 🚀 Quick Start

### Start Development Server
```bash
cd /Users/nameshsingh/Desktop/ashra-portfolio
npm run dev
```
Visit: http://localhost:3000

### Build for Production
```bash
npm run build
npm start
```

## 📝 Key Files to Customize

1. **Portfolio Items** - Edit `/src/data/portfolioData.ts`
2. **Contact Info** - Edit `/src/components/Footer.tsx` and `/src/app/contact/page.tsx`
3. **Brand Colors** - Edit `/src/app/globals.css`
4. **About Content** - Edit `/src/app/about/page.tsx`
