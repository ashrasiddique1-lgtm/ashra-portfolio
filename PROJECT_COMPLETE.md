## ✏️ How to Customize

### 1. Add Portfolio Items
Edit `/src/data/portfolioData.ts`:
```typescript
{
  id: 10,
  title: "Your Design Title",
  category: "Category Name",
  description: "Description",
  image: "image-url",
  year: 2024
}
```

### 2. Update Contact Info
- Email: `/src/components/Footer.tsx` & `/src/app/contact/page.tsx`
- Phone: Same files
- Location: Same files
- Social links: `/src/components/Footer.tsx`

### 3. Change Brand Colors
Edit `/src/app/globals.css` and `/src/app/layout.tsx`

### 4. Add Images
- Place images in `/public` folder
- Reference in portfolio data or pages
